\# Dividend-Adjusted Black-Scholes Pricing Engine



A quantitative finance project implementing the

Black-Scholes option pricing model with dividend support.



\## Features

\- European option pricing with continuous dividend yield

\- European option pricing with discrete cash dividends

\- Full Greeks: Delta, Gamma, Vega, Theta, Rho

\- American option early exercise detection



\## Tech Stack

\- Java 17

\- Apache Commons Math

\- Maven



\## How to Run

mvn compile

mvn exec:java



\## Scenarios

1\. Continuous dividend yield (Merton 1973)

2\. Discrete cash dividend (Forward adjustment)

3\. Option Greeks calculation

4\. American vs European price comparison
